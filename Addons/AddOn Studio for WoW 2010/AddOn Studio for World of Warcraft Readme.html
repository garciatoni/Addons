1. System Requirements for 2010 (v3.0)

1.1 Supported Architectures

  • x86
  • x64

1.2 Supported Operating Systems

  • Windows XP (x86) with Service Pack 3 - all editions except Starter Edition
  • Windows Vista (x86 or x64) with Service Pack 1 - all editions except Starter Edition
  • Windows 7 (x86 or x64)
  • Windows 8 (x86 or x64)
  • Windows 8.1 (x86 or x64)
  • Windows 10 (x86 or x64)
  • Windows Server 2003 (x86 or x64) with Service Pack 2
  • Windows Server 2003 R2 (x86 or x64)
  • Windows Server 2008 (x86 or x64) with Service Pack 2
  • Windows Server 2008 R2 (x64)

2.3 Hardware Requirements

  • Computer that has a 800MHz or faster processor
  • RAM: 512 MB RAM for x86, 1 GB RAM for x64
  • An additional 256 MB RAM if running in a Virtual Machine
  • 20MB of available hard disk space
  • Recommended: DirectX 9-capable video card that runs at 1024 x 768 or higher display resolution

2. System Requirements for 2015 (v4.0)

2.1 Supported Architectures

  • x86
  • x64

2.2 Supported Operating Systems

  • Windows 7 SP 1 (x86 or x64)
  • Windows 8 (x86 or x64)
  • Windows 8.1 (x86 or x64)
  • Windows 10 (x86 or x64)
  • Windows Server 2008 R2 SP 1
  • Windows Server 2012
  • Windows Server 2012 R2

2.3 Hardware Requirements

  • 1.6 GHz or faster processor
  • 1 GB of free RAM
  • An additional 500 MB of free RAM if running on a virtual machine)
  • 4GB of available hard disk space, 5400 RPM or better hard disk drive
  • DirectX 9-capable video card running 1024 x 768 or higher display resolution
